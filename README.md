## :octocat: Setup

### Installation

```sh
# Clone this repo
git clone git@github.com:filipe1309/lotr-nextjs-quiz.git

# Enter project folder
cd lotr-nextjs-quiz

# Install dependencies
yarn
```

### 🏃 Runnning

```sh
yarn dev
# app will run at: http://localhost:3000
```

#### Local

http://localhost:3000

#### API

http://localhost:3000/api/db
